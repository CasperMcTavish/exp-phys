6.2 Numerical Integration

In this checkpoint we will use the simplest numerical integration techniques called First Order Euler which involves iteratively calculating the position and velocity at a set of times ti=iΔtti=iΔt for i=0,1,2,…i=0,1,2,… where we assume that the position and velocity at time ti+1ti+1 depend only on the position and velocity at time titi. In particular if at time titi the particle has

PositionVelocityAcceleration===x→iv→ia→iPosition=x→iVelocity=v→iAcceleration=a→i

then at time ti+1=ti+Δtti+1=ti+Δt the position and velocity of the particle can be approximated by

x→i+1v→i+1==x→i+Δtv→iv→i+Δta→ix→i+1=x→i+Δtv→iv→i+1=v→i+Δta→i

So if we start at time t0=0t0=0 at position x→0x→0 and velocity v→0v→0, then provided we can calculate the force on the particle, and hence the acceleration a→ia→i, and use small enough time steps ΔtΔt, we can iteratively trace out the path by repeated use of the above relations.

6.6 Feedback to Students

The checkpoint is more complex and in particular involves some more involved physics that need to considered first. This problem is not analytically solvable and must be done numerically.

The easiest solutiuon is to follow the route in the Hint and make three functions to set initial_conditions, acceleration and to step_forward the integration by one step.

Then for Checkpoint 5.1, you

  1. Prompt for initial velocity, angle, and ββ.
  2. Set up inital position x=0.0x=0.0, y=0.0y=0.0, vx=vcosθvx=vcosθ, vy=vsinθvy=vsinθ.
  3. form two empy lists to hold the xx and yy values.
  4. Use a while loop to update the xx, yy, vxvx, vyvy values while y≥0y≥0 and record the xx and yy positions in the codelists
  5. plot the x and y lists using plt.plot()

The main strcture is the

XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX

For Checkpoint 5.2 you use the same functions and while loop but you enclose this is a for loop to loop over the angle, so the programme becomes

  1. Prompt for inital velovity and ββ only.
  2. Create two lists to hold the angle and energy ratio data.
  3. Use a for loop to of type for theta in range(0,90): and for each theta set the inital conditions of x=0.0x=0.0, y=0.0y=0.0, vx=vsinθvx=vsinθ, vy=vcosθvy=vcosθ. Append theta to the angle list
  4. Use while loop for y≥0y≥0 and record the velovity when the loop completed; then for the energy ratio and append to list.
  5. when the for loop ends, plot the results with plt.plot()

Here the trick is to nest the while loop to propagate the mass within a for loop running over the angles.
